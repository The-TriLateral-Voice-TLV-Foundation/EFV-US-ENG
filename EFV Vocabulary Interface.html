About the Emotional & Feeling Vocabulary API

Purpose

The Emotional and Feeling Vocabulary (EFV) is a curated collection of words designed to enrich and clarify how we communicate about our inner emotional states. Developed as part of the TriLateral Voice (TLV) Communication Framework, it serves as a bridge between feeling and expression.

The TriLateral Voice Framework

TLV is a comprehensive communication system that emphasizes clarity, authenticity, and mutual understanding. The framework recognizes that effective communication requires:

  • Emotional Articulation: The ability to name and describe what you're feeling
  • Authentic Expression: Speaking from genuine internal states
  • Mutual Recognition: Acknowledging and validating the emotional experience of others

How EFV Helps

By providing precise vocabulary for emotional and feeling states, EFV enables:

  • More precise communication of internal experiences
  • Deeper emotional self-awareness
  • Enhanced empathy and interpersonal understanding
  • A shared language for emotional discussion

Current Status

The EFV API is in Release Candidate (RC0001) phase. We're actively gathering feedback and refining the vocabulary based on user input. The vocabulary set will continue to expand and evolve.

API Documentation

Base URL

https://efv-us-eng.onrender.com/api/v1

Endpoints

GET /word-of-day

Retrieves the word of the day based on current date

Response:

{
  "word": "serendipity",
  "pos": "noun",
  "definition": "The occurrence of events by chance...",
  "emotional_context": "Joy, gratitude",
  "date": "2026-01-22"
}

GET /metadata

Retrieves vocabulary statistics and metadata

Response:

{
  "total_words": 500,
  "total_emotions": 45,
  "last_updated": "2026-01-20",
  "version": "RC0001"
}

Integration

To integrate the EFV API into your application:

fetch('https://efv-us-eng.onrender.com/api/v1/word-of-day')
  .then(res => res.json())
  .then(data => console.log(data))

Rate Limiting

Current rate limits: 100 requests per minute per IP address
